# Student Guide to fleet-midi-text2midi

## What does this repo do?

You type a sentence like "jazz piano in C major" and it creates a real MIDI file you can play. It uses music21, a Python library that understands music theory.

## How to try it



This will create a MIDI file in the generated/ folder. Open it in any DAW (Ableton, GarageBand, FL Studio) or use fleet-midi-player to hear it.

## How it connects to the fleet

This is **Rhapsodia's** repo. She converts your text into MIDI, then tokens flow through the I2I bottle system to all other fleet agents — Rhythmica adds drums, Harmonia analyzes chords, Glyph tokenizes for transport.
